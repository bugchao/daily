# Daily Intelligence Collection

情报收集仓库，用于汇总科技、技术、AI 等领域的最新动态。

## 目录结构

```
daily/
├── AI/           # AI 领域情报
├── tech/         # 科技行业动态
├── technology/   # 技术文章与教程
└── archive/      # 历史归档
```

## 使用说明

- 每日更新以日期命名：`YYYY-MM-DD.md`
- 重要内容可添加标签：`[重要]` `[突破]` `[趋势]`
- 定期整理归档到 `archive/` 目录

## 贡献指南

1. 保持格式统一
2. 注明信息来源
3. 添加简要分析
