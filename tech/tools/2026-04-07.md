# GitNexus - 让代码变成一张"关系网"的神器

## 这玩意儿是干嘛的？

就是个能把 GitHub 代码仓库变成"知识图谱"的工具。你丢给它一个项目，它就给你画出来这个项目里所有文件、函数、变量之间的关系网，还能跟你聊天问答。

重点是：**完全在浏览器里跑，不用装服务器，不用联网**。

## 为什么你需要它？

你肯定遇到过这种情况：

接手一个新项目，打开代码一看，几百个文件，几万行代码。你想知道"这个登录功能到底在哪儿？"、"这个变量被谁用了？"，结果只能靠全局搜索，一个个点开看。

看了三小时，脑子里还是一团浆糊。

有了 GitNexus，你就像拿到了这个项目的"上帝视角"：
- 所有代码关系一目了然
- 想问啥直接问 AI，它会告诉你答案在哪
- 不用担心代码泄露，因为全在你本地浏览器里

## 怎么用？

1. **打开网页** - 访问 GitNexus 网站（或者本地部署）
2. **扔代码进去** - 拖个 GitHub 链接或者 ZIP 文件进去
3. **看图聊天** - 等它生成知识图谱，然后你就可以：
   - 看可视化的代码关系网
   - 跟内置的 AI 聊天："这个项目的核心逻辑是啥？"
   - 点击节点跳转到具体代码

## 说人话的总结

就是个"代码翻译器"，把程序员看不懂的代码，翻译成人类能看懂的关系图。

适合：
- 新人接手老项目（不用再问"这代码谁写的？"）
- 面试前突击学习开源项目（装得像个老手）
- 代码审查（快速找到可疑的地方）

不适合：
- 你的代码只有10行（杀鸡用牛刀）
- 你想偷懒不看代码（AI 能帮你理解，但不能替你思考）
